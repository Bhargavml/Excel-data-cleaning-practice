# Excel Data Cleaning Practice

This project contains practice exercises focused on **data cleaning and transformation using Microsoft Excel**.  
The modules cover common tools and functions used by **data analysts for preparing datasets before analysis**.

---

## Project Overview

The assignment includes several modules that demonstrate how to clean, transform, and organize data using Excel features such as formulas, text tools, and data management techniques.

---

## Modules Covered

### 1. Removing Duplicate and Error Values
Practice creating datasets with duplicate records and identifying common Excel errors.  
Learn how to remove duplicates, clean incorrect values, and verify dataset quality using Excel tools.

---

### 2. Data Cleaning Functions
Use text and numeric functions to standardize and clean messy data.  
Functions practiced include:

- `UPPER`
- `LOWER`
- `PROPER`
- `TRIM`
- `VALUE`
- `LEN`
- `LEFT`
- `RIGHT`
- `MID`

These functions help in **formatting text, removing spaces, extracting characters, and converting text to numbers**.

---

### 3. SEARCH vs FIND Functions
Understand how to locate text within a string using:

- `SEARCH`
- `FIND`

Also practice combining **SEARCH with MID** to extract specific parts of text data.

---

### 4. Go To Special Technique
Learn to quickly select specific types of cells such as:

- Blank cells
- Cells containing formulas
- Constants
- Cells with conditional formatting

This feature is useful for **quick data cleanup and dataset inspection**.

---

### 5. Splitting Data Using Text to Columns
Practice separating combined text values (such as full names) into multiple columns using the **Text to Columns** feature and different delimiters.

---

### 6. Joining Data Using CONCATENATE and (&)
Combine values from different columns using:

- `CONCATENATE`
- `&` (Ampersand operator)

This technique is useful when creating **full names, IDs, or formatted text values**.

---

### 7. Using Find and Replace
Learn how to locate and modify text across a dataset using:

- Find feature
- Replace feature
- Wildcards for pattern matching

This helps quickly correct **repeated or inconsistent values**.
